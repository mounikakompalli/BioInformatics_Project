*******Readme*********************************************************************************************************
The coding is done using C# (console application)
----------------------------------------------------------------------------------------------------------------------
Steps for execution
----------------------------------------------------------------------------------------------------------------------
1)Please copy all the input files(Eg:1NVS.pdb,1AOP.pdb,1AKY.pdb)that you want to test into the debug folder.
path: submitted projects/bin/debug.
2)Double click on the .sln file which will open the code in visual studio.
3)Click on the start button to run the code.
4)You will be prompted to answer a question whether you want alignment.
5)If you enter 'y', you will be prompted to enter the number, names of the input files and their difference factor.
6)After you enter the difference factor the output file with the name output.pdb will be processed.
7)You can check the output file in bin/debug folder.
8)You can right click on the file and open it using chimera.
9)You will be able to view the aligned sequences.
